This is a "space invaders"-inspired game that I'm using for my pairing interview for the Recurse Center.

I'm making a small change to how the aliens move, when one of them hits the wall I want only that alien to move down a row instead of all of them at once. So it's different from the original game in that way.

# How to run

It is a static web project with no server or build steps.

You can just serve this directory on localhost and open /index.html in your web browser.

I've mostly only tested in chromium browsers but this ought to work fine in firefox or safari as well.
